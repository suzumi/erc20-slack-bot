# erc20-slack-bot for testnet(Ropsten)

EthereumのRopstenテストネット上にデプロイしたERC20トークンをSlack上でやりとりするBOT

## 環境変数
```
$ SLACK_TOKEN=
$ ROPSTEN_MNEMONIC=
$ INFURA_ACCESS_TOKEN=
$ CONTRACT_ADDRESS=contract address
$ ISSUER_ADDRESS=owner address
```
